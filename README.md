editor powered by [babylon.js](http://www.babylonjs.com/)

# parameters

## attributes

1. `vec3` position
2. `vec3` normal
3. `vec2` uv

## uniforms

1. `mat4` world
2. `mat4` worldView
3. `mat4` worldViewProjection
4. `mat4` view
5. `mat4` projection
6. `sampler2D` textureSampler
7. `sampler2D` refSampler
8. `float` time
9. `vec3` cameraPosition
10. `sampler2D` picSampler(for post effect)
